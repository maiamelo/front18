Instalação e Integrações

Veja como adicionar o Front18 no seu sistema em menos de 1 minuto e garantir que o rastreamento C.D.N atue em escala.

<!-- Front18 B2B Shield Global -->
<script src="https://front18.b20robots.com.br/js/front18-injector.js"
        data-client="KEY" async></script>

Como instalar nos principais sistemas:

WordPress / Woo

No arquivo header.php (do seu child-theme) ou via plugin "WP Headers".

<head>
    <!-- Outros scripts do WP -->
<!-- Cole o Front18 Shield AQUI -->
    <script src="...front18-injector..."></script>
</head>

Nuvemshop

No painel web, navegue até Configurações > Códigos Avançados.

<!-- Painel Nuvemshop: Rastreamento Cabeçalho -->
<!-- Cole exatamente o código copiado -->
<script src="...front18-injector...js"
        data-client-id="SUA-CHAVE"
        defer></script>

Shopify

Edite sua Loja Virtual. Abra o arquivo central theme.liquid.

<!doctype html>
<html>
    <head>
        {{ content_for_header }}
<!-- Front18 imediatamente antes de fechar head -->
        <script src="..." defer></script>
    </head>

Yampi / Checkout

Na área de desenvolvedor, vá em Scripts e Pixels.
